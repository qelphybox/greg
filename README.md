# greg
Greg is telegram bot, invoice-generator

## Dev

Get the token at https://core.telegram.org/bots#3-how-do-i-create-a-bot.

```
cp .env.sample .env

# paste token from BotFather to .env > TG_BOT_TOKEN=...

make start
```

Into container
```
make install
make s
```
